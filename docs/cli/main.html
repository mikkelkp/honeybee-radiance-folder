honeybee-radiance-folder¶

honeybee folder commands.

honeybee-radiance-folder [OPTIONS] COMMAND [ARGS]...

Options

--version¶

Show the version and exit.

aperture-files¶

Return aperture files.

honeybee-radiance-folder aperture-files [OPTIONS] RADIANCE_FOLDER

Options

-m, --model <model>¶

Optional name for model folder

Default

model

--default, --black¶

Flag to note whether black material files must be returned.

Default

True

-lf, --log-file <log_file>¶

Optional log file to export the output. By default this will be printed to stdout

filter-folder¶

Filter files in a folder.

Args:
folder: path to folder.
pattern: a regex pattern to filter files based on. The pattern must be a raw
string with no quotes. The command compiles the string to a regex pattern.
honeybee-radiance-folder filter-folder [OPTIONS] FOLDER PATTERN

Options

-lf, --log-file <log_file>¶

Optional log file to export the output. By default this will be printed to stdout

Arguments

FOLDER¶

Required argument

PATTERN¶

Required argument

filter-file¶

Filter a list fo JSON objects based on value for a specific key.

Args:
input-file: Path to input JSON file. Input JSON file should be an array of
JSON objects.
pattern: Two string values separated by a :. For example group:daylight
will keep/remove the objects when the value for group key is set to daylight.
honeybee-radiance-folder filter-file [OPTIONS] INPUT_FILE [PATTERN]

Options

-of, --output-file <output_file>¶
--keep, --remove¶

A flag to switch between keeping the objects or removing them from the input list.

Default

True

INPUT_FILE¶

Required argument

PATTERN¶
